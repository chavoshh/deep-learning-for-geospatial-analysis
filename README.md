# Deep Learning for Geospatial Analysis

An exercise to build U-Net models in PyTorch for satellite imagery, crop mapping, change detection, building segmentation and LiDAR. The complete guide and all rights of this educational project belongs to www.udemy.com under the name: Fundamentals of GeoAI: Deep Learning for Geospatial Analysis.

## Setup

This project uses [uv](https://docs.astral.sh/uv/) for Python and
dependency management.

```bash
git clone https://github.com/Chavoshh/Deep-Learning-for-Geospatial-Analysis.git
cd DL-Geospatial-Analysis
uv sync
```
